『ソースプログラム』について


　本ディスクの『SOURCE』フォルダには、本文で示したすべてのソースプログラムが格納されています。

　ソースプログラムは、各章ごとのフォルダに分類されて格納されています。たとえば、第5章のソースプログラムは『chap05』フォルダに入っています。

　なお、コンパイル・実行をしやすくするために、本文p.114のList 3-8 "Member.h"とp.115のList 3-9 "Member.c"は、chap03だけでなく、chap09とchap10にも同じものが格納されています。同様に、本文p.146のList4-1 "IntStack.h"とp.147のList 4-2 "IntStack.c"は、chap04だけでなく、chap05とchap06にも同じものが格納されています。
